Creates a new instance of AdTonosVastUrlBuilder. Each time a new URL to VAST is needed it must be generated using the new builder. This method can only be called after the start method has been executed and finished. To check the execution status of the start method you can use isStarted. Builder is needed to enter the necessary additional data. One of the relevant elements is the AdTonos key.

The method returns whether the startup process has finished. The startup process consists of a call to the start method and the granting of system permissions by the user. The method can be called before initialize is executed, but then it returns false.

Content copied to clipboard

This method can be used when the process of granting consents is more complicated to save the current values. After the start method is called, the saveConsents method updates the data collection. It can be used for example to revoke the consents on the settings screen. Calling saveConsents alone is not sufficient to start the library.

Content copied to clipboard

The library requires certain permissions and user consents for data collection. Therefore, one of the initial screens should show the user the terms and conditions containing the necessary information about the use and processing of personal data. This is done when the user first interacts with the application, so this is a good time to call the start method. Additionally the start method savesConsents.

The method takes the AdTonosConsent consent flag as a parameter and then asks the user for the system permissions necessary for the application to work. This process is automated. The start method must be called every time the application starts. Additionally, the loadLatestConsents method can be used, which returns the most recently granted consents.
