The library requires certain permissions and user consents for data collection. Therefore, one of the initial screens should show the user the terms and conditions containing the necessary information about the use and processing of personal data. This is done when the user first interacts with the application, so this is a good time to call the start method. Additionally the start method savesConsents. The method takes the AdTonosConsent consent flag as a parameter and then asks the user for the system permissions necessary for the application to work. This process is automated. The start method must be called every time the application starts. Additionally, the loadLatestConsents method can be used, which returns the most recently granted consents. Below is an example of how this can be done:

var consents = AdTonosSDK.loadLatestConsents()
if (consents == null) {
  //show terms and conditions, delegate result from terms and conditions to other method e.g.
  //onPrivacyPolicyPopupClosed
  return
}
AdTonosSDK.start(applicationContext, consents)
