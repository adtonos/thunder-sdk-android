AdTonos Link Generator SDK

Url generator for personalized VAST audio advertisement.

Singleton object containing a set of methods for generating personalized VAST URL

Installation

Add the following code to the application manifest between the application tags:

<activity android:name="com.siroccomobile.linkgenerator.adtonos.ui.AdTonosActivity" android:theme="@style/AppTheme.Transparent" />
Content copied to clipboard

Usage:

  1. Call initialize at initialization time of the app.

  2. Call start

  3. Call createBuilder and fill key with AdTonosVastUrlBuilder.setAdTonosKey

  4. On the instance of AdTonosVastUrlBuilder call AdTonosVastUrlBuilder.build method to get URL

  5. Repeat step 4 every time VAST link is required

  6. When application is being destroyed call dispose method to free all resources

Creates a new instance of AdTonosVastUrlBuilder. Each time a new URL to VAST is needed it must be generated using the new builder. This method can only be called after the start method has been executed and finished. To check the execution status of the start method you can use isStarted. Builder is needed to enter the necessary additional data. One of the relevant elements is the AdTonos key.
